# Lab 2: vSAN Storage Policies Configuration

## Overview
This lab covers vSAN storage policies configuration and management.

## Learning Objectives
- Understand vSAN storage policies
- Configure VM storage policies
- Monitor policy compliance

## Prerequisites
- Completed Lab 1: Basic Setup
- vSAN cluster configured

## Lab Tasks

### Task 1: Default Storage Policy Analysis
1. Review default vSAN storage policy
2. Understand policy components
3. Analyze current VM assignments

### Task 2: Custom Storage Policy Creation
1. Create custom storage policies
2. Configure failure tolerance
3. Set performance requirements

### Task 3: VM Storage Policy Assignment
1. Assign policies to VMs
2. Monitor policy compliance
3. Troubleshoot policy violations

## Expected Results
- Custom storage policies created
- VMs assigned appropriate policies
- Policy compliance monitored
