# Lab Environment Setup

## Overview
This guide covers the setup of lab environment for vSAN 8 learning.

## Prerequisites
- VMware vSphere 8 licenses
- Compatible hardware or nested virtualization
- Network configuration
- Storage requirements

## Environment Components
- ESXi hosts (minimum 3)
- vCenter Server
- Shared storage
- Network infrastructure

## Setup Steps
1. Install ESXi hosts
2. Deploy vCenter Server
3. Configure networking
4. Create vSAN cluster

## Validation
- Verify cluster health
- Test basic functionality
- Validate performance
- Document configuration
