# Lab 4: vSAN Stretched Cluster Configuration

## Overview
This lab covers vSAN stretched cluster deployment and configuration.

## Learning Objectives
- Understand stretched cluster architecture
- Configure stretched cluster
- Implement disaster recovery scenarios

## Prerequisites
- Multiple sites available
- Network connectivity between sites
- Witness host configured

## Lab Tasks

### Task 1: Stretched Cluster Planning
1. Design stretched cluster architecture
2. Plan network requirements
3. Configure witness host

### Task 2: Stretched Cluster Deployment
1. Configure stretched cluster
2. Validate connectivity
3. Test failover scenarios

### Task 3: Disaster Recovery Testing
1. Simulate site failure
2. Validate automatic failover
3. Test manual recovery procedures

## Expected Results
- Stretched cluster configured
- Failover scenarios tested
- Recovery procedures validated
